# CHANGELOG

本プロジェクトの変更履歴です。

---

## [1.1.0] — 2026-03-11

### 土木・法面工事向け本格アップグレード

#### 追加
- 工事番号・元請会社名・現場代理人名フィールドを追加
- 天候入力を午前/午後の独立した2系統に分離
- 作業員情報を職種別（土木作業員/重機オペレーター/交通誘導員/その他）に細分化、合計自動計算
- 使用重機・機械の動的リスト（機械名＋台数、行の追加/削除）
- 工種セレクトボックス（法面工事/土工/排水工/法枠工/アンカー工/吹付工 等）
- 作業進捗率スライダー（0〜100%、5%刻み）
- 明日の作業予定フィールドに音声入力ボタンを追加
- 安全管理セクション（朝礼/KY活動/ヒヤリハット 有無トグル、ヒヤリハット内容入力）
- 品質管理セクション（検査・立会、写真撮影枚数、気温）
- Googleドライブ保存機能：「建設日報」フォルダを自動検索または作成して保存
- 保存完了後にドライブリンクをその場に表示
- 印刷ボタン（A4用紙設定で新規ウィンドウ印刷）

#### 変更
- AI日報プロンプトを土木・法面工事専門に全面改訂（ですます調・A4一枚・専門用語使用）
- UIをカード分割・セクション化して見通しを改善
- 音声モードの識別ヒントテキストを緑色で強調

## [1.0.1] — 2026-03-11

### iOS Safari 音声入力対応修正

#### 修正
- `canMediaRecord` 判定が iOS Safari の `file://` プロトコルで常に false になる問題を修正
- セキュアコンテキスト判定を `isSecureCtx` 変数として独立、`location.protocol` / `hostname` / `window.isSecureContext` を組み合わせて確実に判定
- iOS + 非HTTPSの場合に専用メッセージ「HTTPS配信が必要」を表示
- iPhoneユーザー向けにHTTPS配信方法（ローカルサーバー / Netlify Drop）のガイドカードを追加

## [1.0.0] — 2026-03-11

### 初回リリース

#### 機能
- スマートフォン最適化シングルHTMLファイルアプリ
- 現場名・作業日・天候・作業員数・作業内容・特記事項の入力フォーム
- Web Speech API によるリアルタイム音声入力
- OpenAI Whisper API によるフォールバック音声入力（iOS Safari向け）
- ChatGPT API（gpt-4o）による日報自動生成
- 生成日報のプレビュー表示
- テキストファイルダウンロード
- クリップボードコピー
- Googleドライブ保存（OAuth2認証）
- APIキー・設定のlocalStorage保存機能
- APIキー未設定時のアラートバナー
